## 单词和字符间距

`word-spacing`和`letter-spacing`属性分别用来设置单词之间的间距和字符之间的间距。

下面是两种间距的区别：

```html
<style>
    .p1 {
        word-spacing: 30px;
    }
    .p2 {
        letter-spacing: 30px;
    }
</style>
<div class="p1">we are family.</div>
<div class="p2">we are family.</div>
```
